# Aerobatic-Planner

An autonomous aerobatic system that is capable of complex flight maneuvers, typically achievable only by professional pilots.

This repository is forked from [ZJU-FAST-Lab/Aerobatic-Planner](https://github.com/ZJU-FAST-Lab/Aerobatic-Planner) **with modifications for** Gazebo simulation and real-world experiments.
